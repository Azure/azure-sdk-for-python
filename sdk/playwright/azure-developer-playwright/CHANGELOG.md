# Release History

## 1.0.0b1 (2026-04-06)

### Features Added

  - Initial preview release of the Azure Developer Playwright client library for Python
  - Based on Azure Developer Playwright data-plane API version **2026-04-01-preview**
  - Support for managing access tokens (create, get, delete, list)
  - Support for managing test runs (create or update, get, list)
  - Support for managing browser sessions (get, list)
  - Support for getting remote browsers for Playwright script execution
