# Sample Files

This directory should contain sample files for testing:

- `sample_invoice.pdf` - Sample invoice for document analysis

## Where to Get Sample Files

You can use any PDF, image, or video file for testing. For example:
- Download a sample invoice from: https://github.com/Azure-Samples/cognitive-services-REST-api-samples/tree/master/curl/form-recognizer
- Or use your own test files

## Usage

Samples expect files in this directory:
- `content_analyzers_analyze_binary.py` - Uses `sample_invoice.pdf`
- `content_analyzers_analyze_binary_raw_json.py` - Uses `sample_invoice.pdf`

Place your test files here and update the sample code if needed.
